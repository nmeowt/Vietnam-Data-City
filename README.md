# Getting started
- Clone the repository
```
git clone --depth=1 https://github.com/nmtrangg99/Vietnam-Data-City.git <project_name>
```
- Install dependencies
```
cd <project_name>
npm install
```
- Create database ``vietnam-city``

- Insert to database 
```
GET: /api/v1/insert
```
